## Getting Started

平台组件库

Global install lerna
$ npm i lerna@6.6.2 -g

Install dependencies,

```bash
$ npm run init
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs
```

Build library via `father-build`,

```bash
$ npm run build
```
